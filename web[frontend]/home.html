How to Use

  1. Register with your mail id.
  2. Click the **Choose File** button below to upload an image containing Hindi text.
  3. Press the **Process Image** button to start text extraction.
  4. Wait for the system to analyze the image and extract the text.
  5. Copy and use the extracted text as needed.
  6. Provide feedback to help improve the system.
